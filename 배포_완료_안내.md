## 🚀 quest-k.com에서 보려면

### **방법 1: Vercel 웹사이트에서 재배포 (5분)**

1. **Vercel 로그인**: https://vercel.com
2. **k-quest 프로젝트 선택**
3. **Deployments 탭** 클릭
4. **최신 커밋 확인** ("준비중 페이지 완성 및 빌드 수정")
5. **자동 배포 확인** 또는 **"Redeploy" 클릭**

---

### **방법 2: Vercel CLI로 배포 (즉시)**

**명령어:**
```powershell
npx vercel --prod --yes
```

이 명령어를 실행하면:
1. Vercel 로그인 요청 (브라우저 열림)
2. 프로젝트 선택
3. 즉시 배포 시작
4. URL 생성
