# 개인정보 보호 가이드

## 🔐 K-Quest 개인정보 최소화 전략

### 법적 요구사항 vs 개인정보 보호

한국 전자상거래법에 따라 **반드시 공개해야 하는 정보**:
- ✅ 상호명
- ✅ 대표자 성명
- ✅ 사업자등록번호
- ✅ 통신판매업 신고번호
- ✅ 영업소 소재지 (상세 주소 필수 아님)
- ✅ 연락처 (전화 또는 이메일)

---

## 📱 **현재 K-Quest 설정** (개인정보 최소화)

### 일반 사용자가 보는 화면:
1. **메인 페이지**: 사업자 정보 전혀 없음
2. **Footer**: 
   - "© 2025 K-Quest"만 표시
   - 이용약관, 개인정보처리방침 링크
   - 맨 아래 작은 글씨로 "사업자 정보" 링크 (눈에 안 띔)

### `/company` 페이지 (법적 의무):
- **주소**: "경기도 의정부시"만 표시 (상세 주소 숨김)
- **연락처**: 이메일만 표시 (전화번호 없음)
- **기타**: 법적으로 필요한 사업자등록번호, 통신판매업 신고번호만

---

## ✅ **추가 개인정보 보호 방법**

### Option 1: 대표자 이름도 숨기기 (⚠️ 법적으로 위험)
```tsx
owner: '포텐타로 대표'  // 이름 대신 상호 + "대표"
```

### Option 2: 사업자 정보 페이지를 robots.txt에서 제외 (검색엔진 차단)
```
User-agent: *
Disallow: /company
```

### Option 3: 문의는 이메일만 받기 (현재 설정)
- ✅ 전화번호 없음
- ✅ 이메일: wkrwkr777@gmail.com

### Option 4: 상호명을 "포텐타로" 대신 영문 "Potentaro"로 표시

## 🎯 **현재 K-Quest는 이렇게 되어 있습니다**:

### ✅ 일반 사용자 화면
- 홍보/마케팅 페이지만 보임
- 개인정보 전혀 노출 안 됨

### ✅ 법적 요구사항 준수
- `/company` 페이지에만 필수 정보
- Footer 맨 아래 작은 링크로만 접근 가능
- 주소: 시 단위까지만 표시

---

## 💡 **결론: 현재 설정이 Best Practice입니다!**

- 법적 의무는 지키면서
- 개인정보는 최소한만 공개
- 일반 사용자는 사업자 정보를 볼 필요도 없고 찾기도 어려움
