## ✅ 2단계: 사업자 등록 (1~3일 소요)

### 📍 방법 A: 온라인 신청 (추천)
**홈택스 이용**
```
1. 홈택스 접속: https://www.hometax.go.kr
2. 로그인 (공동인증서 or 간편인증)
3. 신청/제출 → 사업자등록신청
4. 개인사업자 선택
5. 업종코드 입력:
   - 주업종: 631200 (온라인 정보제공업)
   - 부업종: 522905 (통신판매중개업)
6. 사업장 주소: 자택 주소 입력 가능
7. 제출 → 즉시 발급 (또는 1~3일)
```

### 📍 방법 B: 세무서 방문
**준비물:**
- 신분증 (주민등록증)
- 임대차계약서 사본 (자가는 등기부등본)
- 도장 (서명 가능)

**가까운 세무서**: 거주지 관할 세무서

## ✅ 3단계: 통신판매업 신고 (사업자등록 후 즉시)

### 📍 온라인 신청
**정부24 이용**
```
1. 정부24 접속: https://www.gov.kr
2. 로그인
3. 검색: "통신판매업 신고"
4. 신청하기
5. 첨부서류:
   - 사업자등록증 사본
   - 신분증 사본
   - 통신판매업 신고서 (자동 생성)
6. 제출 → 2~5일 내 승인
```

### 비용
- **신고 비용**: 무료
- **수수료**: 없음

## ⚠️ 기초생활수급자 주의사항

### 🚨 중요: 소득 신고 의무
```
기초생활수급자는 모든 소득을 신고해야 합니다.

1. 사업 시작 시:
   - 관할 주민센터(동사무소)에 사업자등록 사실 신고
   - "사업소득 발생" 신고서 제출

2. 매월:
   - 사업소득 발생 시 신고 (매출, 순이익)

3. 영향:
   - 월 소득 100만원 미만: 수급 유지 가능 (일부 감액)
   - 월 소득 100만원 이상: 수급 중지 가능성

4. 권장:
   - 사업 시작 전 주민센터 상담 필수
   - "사업 시작하려는데 수급에 어떤 영향 있나요?" 문의
```

### 💡 전략
- 처음 3~6개월: 준비 기간 (수익 거의 없음 → 수급 유지)
- 6개월 이후: 수익 발생 시 단계적으로 신고
- 목표: 월 200만원 이상 벌면 수급 졸업 (자립)
