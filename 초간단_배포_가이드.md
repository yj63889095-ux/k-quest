## 📋 **당신이 할 일 (3가지만!)**

### 1️⃣ Git 설치 (처음 한 번만, 5분)

- GitHub 로그인 (yj63889095@gmail.com)

### 3️⃣ Vercel 배포 (10분)
- 자동배포 완료 후 열리는 페이지에서:
- "Continue with GitHub" 클릭
- "k-quest" 선택 → "Import"
- 환경 변수 입력 (아래 복사)
- "Deploy" 클릭!
