# 🚀 K-Quest 배포 - 정말 쉬운 버전 (클릭만 하면 됨!)

## 😊 걱정 마세요! 제가 최대한 쉽게 만들었어요!

브라우저에 열린 **"배포도우미.html"** 파일을 보면서 따라하시면 됩니다!

---

## 📋 **전체 과정 요약** (30분이면 끝!)

### ✅ STEP 0: Git 설치 (5분) - 처음 한 번만!
1. [Git 다운로드](https://git-scm.com/download/win) 클릭
2. 다운로드된 파일 실행
3. 계속 "Next" 클릭 (기본 설정으로!)
4. "Install" 클릭
5. 설치 완료!

### ✅ STEP 1: GitHub 저장소 만들기 (3분)
1. 배포도우미 페이지에서 **"1️⃣ GitHub 저장소 만들러 가기"** 버튼 클릭
2. Repository name에 `k-quest` 입력
3. Public 선택
4. "Create repository" 버튼 클릭
5. 끝!

### ✅ STEP 2: 코드 업로드 (5분)
1. 배포도우미 페이지에서 **"📋 명령어 복사하기"** 버튼 클릭
2. PowerShell 열기 (윈도우 검색에서 "PowerShell")
3. 붙여넣기 (Ctrl+V)
4. YOUR_USERNAME만 본인 GitHub 이름으로 바꾸기
5. Enter 키!

### ✅ STEP 3: Vercel 배포 (10분)
1. 배포도우미 페이지에서 **"2️⃣ Vercel 배포하러 가기"** 버튼 클릭
2. "Continue with GitHub" 클릭
3. "k-quest" 찾아서 "Import" 클릭
4. 환경 변수 복사 버튼 클릭해서 붙여넣기
5. "Deploy" 버튼 클릭!

### ✅ STEP 4: 도메인 연결 (15분)
1. Vercel에서 Settings → Domains
2. `quest-k.com` 입력하고 Add
3. 도메인 등록 업체에서 DNS 설정
4. 완료!
